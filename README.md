# Schema, vecka 45
###### Fullstack Webapplikation med TypeScript, vecka 1 av 7

## Introduktion

Den här veckan går vi vidare till att använda React tillsammans med TypeScript. Vi lär oss hur man sätter upp ett projekt med Vite och hur TypeScript hjälper oss skriva säkrare och mer förutsägbar React-kod.

Vi tittar på hur man typar props och state, använder interfaces för komponenter och hur typningen fångar fel redan under utvecklingen. Dessutom går vi igenom exempel på hur man hanterar events och funktionella komponenter med TypeScript.

## Mål för veckan:

1. Kunna arbeta med AI-assisterad utveckling och kritiskt granska koden
2. Kunna utveckla koden tillsammans med AI och modifiera koden efter behov
3. Kunna typa upp props med interfaces / types
